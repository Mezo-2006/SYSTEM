# Presentation

This folder contains a lightweight HTML/CSS/JS slideshow representing the project.

- Open `presentation/index.html` in a browser to view.
- Customize slides in `index.html`, styles in `style.css`, and behavior in `script.js`.

Suggestions:
- Add screenshots (place under `presentation/assets/` and reference in slides)
- Embed recorded demos with `<video>` tags
- Convert to Reveal.js or Deckset for advanced features
