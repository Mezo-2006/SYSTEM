Switch-Case Compiler

Educational compiler demonstrating compilation phases and visualizations

Prepared from repository: SwitchCaseCompiler

Use arrow keys or click to navigate · press `H` for help

Overview

  • Educational compiler for a simplified language focused on switch-case statements
  • Demonstrates lexer, parser (rightmost derivation), semantic analysis, TAC, optimizations, codegen
  • Includes a Qt-based GUI with multiple visualization panels

Key Features

  • Derivation Step Viewer — animated rightmost derivation
  • Parse Tree & Token / Symbol Tables
  • Three-Column Trace: Source / TAC / Assembly
  • Optimizations: constant folding, DCE, CSE

Architecture

  1. Lexical Analysis (scanner + symbol table)
  2. Syntax Analysis (explicit derivation parser)
  3. Semantic Analysis & TAC generation
  4. Optimization passes and target code generation

Usage & Build

See the project's README and BUILD instructions. Quick notes:

  • CMake-based build; Qt required for the GUI
  • CLI available for automated testing and regression

Demo Example

switch (x) {
  case 1: y = x + 2; break;
  case 2: y = x * 3; break;
  default: y = 0; break;
}
