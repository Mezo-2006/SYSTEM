⚡ Optimization Techniques

🔢 Constant Folding

  • Compile-time expression evaluation
  • Reduces runtime computation
  • Example: 2 + 3 → 5

🗑️ Dead Code Elimination

  • Removes unreachable code
  • Eliminates unused variables
  • Reduces code size

🔄 Common Subexpression Elimination

  • Identifies repeated computations
  • Reuses calculated values
  • Improves performance

📋 Copy Propagation

  • Replaces variable copies
  • Reduces memory operations
  • Simplifies data flow

💪 Strength Reduction

  • Replaces expensive operations
  • Multiplication → Addition
  • Division → Shift operations

🔍 Algebraic Simplification

  • Applies mathematical identities
  • x * 1 → x, x + 0 → x
  • Reduces instruction count
